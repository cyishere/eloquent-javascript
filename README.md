# 关于

本人正在学习 JavaScript，并且希望通过翻译《[Eloquent JavaScript](https://eloquentjavascript.net/index.html)》来更好地理解和巩固知识。

翻译如果不当之处，欢迎指正！

可以在`pages/`文件夹中查看我正在翻译的内容。

也可以通过 clone 本 repo 来在本地运行此 Next.js 站点，来更好地查看译本。

```bash
# === 安装 ===

cd eloquent-js-zh

yarn install

# 或者

npm install

# === 运行 ===
yarn dev

# 或者

npm run dev
```
